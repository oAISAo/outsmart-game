# Environment Setup Guide

## Purpose
This document ensures that all developers (and AI copilots) work in a consistent and reliable environment for the **Real-Time Sync App** project built with **Angular + Ionic + Capacitor**.

---

## 1️⃣ System Requirements

| Tool | Recommended Version | Notes |
|------|---------------------|-------|
| **Node.js** | 20.x (LTS) | Use a stable LTS version for best compatibility |
| **npm** | 10.x | Comes with Node, ensure it’s up to date |
| **Angular CLI** | 17.x or newer | Use the same major version as the project |
| **Ionic CLI** | 7.x | Required for running and building the app |
| **Capacitor CLI** | 6.x | Used for native builds |
| **VS Code** | Latest | Preferred IDE with Copilot integration |
| **Git** | Latest | For version control and collaboration |

---
